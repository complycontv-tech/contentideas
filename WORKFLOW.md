# Launchpad Marketing — Content Ideas 🇸🇰

## Workflow
1. Social Content Creator generates 5-10 fresh ideas daily using the Nika x Uhol matrix
2. Ideas saved as `.md` files in `ideas/YYYY-MM-DD/` with proper frontmatter
3. `index.html` auto-updates to show all ideas
4. Changes committed and pushed to GitHub daily

## Structure
```
ideas/
├── Marketing/
├── Gastro/
└── Personal Brand/
index.html — browse ideas online
daily-generator.js — generates fresh ideas
```

## Branch Strategy
- `main` — production (published via GitHub Pages)
- Feature branches for experimental content formats
